# cashAI-skills

CashAI automation skills and tools by Sal Tarango.

## Skills

### gemini-image-gen
Generate images using Google Gemini Nano Banana models via the REST API.
- Supports Nano Banana, Nano Banana 2, Nano Banana Pro, and Imagen 4
- PowerShell script for Windows
- Requires GEMINI_API_KEY with billing enabled
